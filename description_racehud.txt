Race HUD inspired by DeFRaG World Cup 2017 Movie: https://www.youtube.com/watch?v=cCO9QuV-cgM

- Customizable font and text settings
- Adjustable landing effect settings
- Elements can be moved around with Brandon's Hud Editor

Requires [url=http://steamcommunity.com/sharedfiles/filedetails/?id=1135362632]GoaHud[/url] to be installed and up to date (version r45 or later).

Other GoaHud widgets:
- [url=http://steamcommunity.com/sharedfiles/filedetails/?id=1135362632]GoaHud_EmojiSupport[/url]

GitHub: https://github.com/GoaLitiuM/GoaHud

Note: All Reflex Arena addons must be installed via in-game addons menu instead of using the workshop subscription system.
